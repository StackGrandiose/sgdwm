## Patches
- autostart scripts
- bar padding
- stacker
- tag preview
- xresources
- hide vacant tags
- fullgaps

## Requirements
- picom, dwmblocks, nitrogen, redshift

## Setup
- Copy `autostart/autostart.sh` and `autostart/autostart_blocking.sh` to `~/.local/share/dwm/`
- `sudo make clean install`
